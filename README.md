Программа имеет возможности: 
- регистрации и запоминания пользователей (register);
- авторизацией пользователей (login);
- сокращения полных URL до коротких с лимитами по использованиям, ограниченным временем действия - сутки (create);
- сохранения списка ссылок (mylinks). 

Для установки программы, необходимо:
- скачать все файлы и поместить их в одну папку;
- открыть через IDE систему папку с файлами и запустить программу.
  
Авторизация новых пользователей возможна через команду login без необходимости выходить из предыдущего профиля.
Благодаря запоминанию пользователей, программа не даст возможности зарегистрировать пользователя с уже существующим логином.
Ссылки генерируются уникально для каждого пользователя.
